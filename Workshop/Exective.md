# 経営層の方向けのWorkshop

会社や組織の意思決定層の方を対象にしたPromptのサンプルです。

同じチャットに連続で入力をしてください。それぞれのチャットに名前をつけておくと、後で見返すときに便利です。

Microsoft 365 Copilotを活用します。

# Step 1. 課題の明確化   

このStepは、使えるようでしたら「リサーチツール」で行うと更に深く広く課題を洗い出すことができます。
リサーチツールを使う際には:
- [新しいチャット]に移動
- [ツール]から、[@ その他のエージェントを探す] を選択
- [リサーチツール] を選択して開始してください。

出力されたら「リサーチツール」の右側の「×」を押して、通常のCopilotのチャットに戻ります。

```text
# 課題の明確化

以下の観点から、我が社が抱える業務課題を箇条書きで整理してください。また、生成AIが寄与できそうな領域をマークしてください。作成した一覧を「課題整理」という名前を付けてください。
・業務効率化の必要性
・人材不足
・情報共有の非効率
・コスト構造
・競争力の課題
```

#  Step 2. 情報収集・現状分析

```text
# 情報収集・現状分析

この「課題整理」について、
「自動化可能性」「生成AIによる効率化可能性」「リスク」
を評価して1-5の5段階（5=高い, 1=低い）で分類してください。
```


## Step 2.1. 仮説としての定量化

```text
生成AIで効率化した場合の工数削減の試算を行ってください。
1件あたりの平均時間は 30分 と仮定してください。
```

# Step 3. 生成AI活用案の作成

```text
# 生成AI活用案の作成

「情報収集・現状分析」をもとに、生成AI活用のユースケースを5〜10個提案してください。

分類は以下でお願いします：
・横断業務効率化
・部門特化AI
・AIエージェント活用
・ガバナンスやセキュリティ強化
```

## Step. 3.1. 活用案の詳細化

```text
各ユースケースについて、期待される効果・必要リソース・リスクを表形式でまとめてください。
```

# Step 4. 選択肢の評価・優先順位付け

```text
# 選択肢の評価・優先順位付け

このユースケース一覧を、以下の評価軸でスコア化してください（1〜5点）
・効果
・実現可能性
・コスト
・リスクの低さ

総合スコアを計算し、優先順位順に並び替えてください。
```

# Step 5. 経営方針案の作成

このStepは、使えるようでしたら「リサーチツール」で行うと更に深く広く課題を洗い出すことができます。

```text
# 経営方針案の作成

これまでのワークショップで作成した情報をもとに、我が社の「生成AI活用方針（ドラフト）」を作成してください。

構成は以下でお願いします。

1. 活用の目的（業務効率化・競争力向上）
2. 優先導入する部門
3. 導入するAIの種類（汎用生成AI、部門特化AI、AIエージェント）
4. ガバナンス・リスク対策（社内ガイドライン案も含む）
5. KPI とモニタリング方法
6. 今後の展開方針（全社展開ロードマップ）
```
